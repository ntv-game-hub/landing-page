# Kho Game Vui

Static landing page hiển thị các game đang deploy trên Render.

## Games

- Caro Multiplayer: https://caro-multiplayer.onrender.com
- Word Guessing: https://ntv-word-guessing.onrender.com
- Bingo Vui: https://ntv-bingo.onrender.com
- Vòng Quay Thử Thách: https://ntv-challenge-wheel.onrender.com

## Run

Mở trực tiếp `index.html` trong browser hoặc deploy thư mục này lên bất kỳ static hosting nào.
